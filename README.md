# Cools Cats

A full-stack app that displays cat photos using [TheCatsAPI](https://docs.thecatapi.com/).
See the [demo here](#)

## Libraries used

- React-router-dom
- React-bootstrap / Bootstrap
- Axios
- Context API (for state management)
- Styled-components
- Typescript (recommended)
- Eslint
